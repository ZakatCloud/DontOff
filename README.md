# DontOff
Низкоуровневый скрипт на C++ который не позволяет системе ни в коем случае прервать работу. Для ноутбуков со строгим спящим режимом или отключением интернета при засыпании экрана.

Скачать : https://github.com/ZakatCloud/DontOff/releases

- По факту все это можно сделать в системе. Но я решил сделать для этого скрипт, тем более даже после настройки мой ноутбук при закрытии крышки обрубал доступ в интернет - с этим скриптом все иначе.
